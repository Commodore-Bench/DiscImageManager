Command line options

Long form				Short form
--insert:<filename>			-i		- Opens image file <filename>
--new:<format>				-n		- Creates a blank image file for format <format> with <T> tracks (see below for format numbers) 
--add:<filename>[<destination>]		-a		- Adds files to the open image in <destination> directory. Assumes drive 0 and root if nothing selected or not found.
--delete:<filename>			-d		- Deletes file/directory from inside image
*--rename:<oldfilename>:<newfilename>	-r		- Renames <oldfilename> to <newfilename>
--save:[<filename>]			-s		- Saves image file as <filename>
*--create:<directory>			-c		- Creates a new directory, in the image.
*--extract:<filename>			-e		- Extracts file(s) to local path.
*--title:<disc title>			-t		- Sets the title for the disc
*--title1:<disc title>			-t1		- Sets the title for the disc, side 1
*--opt:<opt>				-o		- Sets the boot option for the disc
*--opt1:<opt>				-o1		- Sets the boot option for the disc, side 1

The image will be saved, if changed, after all operations are complete, using the filename specified with --insert or --save, or Untitled with --new.
