Command line options

Long form				Short form
--insert:<filename>			-i		- Opens image file <filename>
--new:<format>				-n		- Creates a blank image file for format <format> with <T> tracks (see below for format numbers) 
--add:<filename>[|<destination>]	-a		- Adds files to the open image in <destination> directory. Assumes drive 0 and root if nothing selected or not found.
--delete:<filename>			-d		- Deletes file/directory from inside image
--rename:<oldfilename>|<newfilename>	-r		- Renames <oldfilename> to <newfilename>
--save:[<filename>]			-s		- Saves image file as <filename>
--create:[<newdir>]|[<parentdir>]	-c		- Creates a new directory, <newdir>, in the image as a child of <parentdir>.
--extract:<filename>			-e		- Extracts file(s) to local path.
--title:<disc title>			-t		- Sets the title for the disc
--title1:<disc title>			-t1		- Sets the title for the disc, side 1
--opt:<opt>				-o		- Sets the boot option for the disc
--opt1:<opt>				-o1		- Sets the boot option for the disc, side 1
--cmdfile:<filename>			-f		- Reads and executes commands held in a text file
--keepopen				-k		- Keeps the application open after parsing the command line options

Notes:
1 Still to be written.
2 There is currently only minor error checking.
3 The image will be saved, if changed, after all operations are complete, using the filename specified with --insert or --save, or Untitled with --new.
4 Any unrecognised commands, or insufficient number of parameters, will be ignored.

<format> can be:
DFSS	: Acorn DFS single sided 80 track
DFSS40	: Acorn DFS single sided 40 track
DFSD	: Acorn DFS double sided 80 track
DFSD40	: Acorn DFS double sided 40 track
WDFSS	: Watford DFS single sided 80 track
WDFSS40	: Watford DFS single sided 40 track
WDFSD	: Watford DFS double sided 80 track
WDFSD40	: Watford DFS double sided 40 track
ADFSS	: Acorn ADFS S
ADFSM	: Acorn ADFS M
ADFSL	: Acorn ADFS L
ADFSD	: Acorn ADFS D
ADFSE	: Acorn ADFS E
ADFSE+	: Acorn ADFS E+
ADFSF	: Acorn ADFS F
ADFSF+	: Acorn ADFS F+
C1541	: Commodore 1541
C1571	: Commodore 1571
C1581	: Commodore 1581
AMIGADD	: Commodore Amiga DD
AMIGAHD	: Commodore Amiga HD
CFS	: Acorn Cassette Filing System

<opt> can be:
0 or none	:	*OPT4,0
1 or load	:	*OPT4,1
2 or run	:	*OPT4,2
3 or exec	:	*OPT4,3
